- Mac

## Instalação
Instalando globalmente:

    npm install -g tomanocupacero

## Running
Run command:

    tomanocupacero
    
## Rodando com [npx](https://www.npmjs.com/package/npx)
Rode o comando:

    npx tomanocupacero
